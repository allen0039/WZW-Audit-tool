# 审计实训万能填表助手 (WZW-Audit-tool)

智能审计实践教学平台（http://10.18.0.178:9350）的 Tampermonkey 脚本，自动识别 SpreadJS 表格，从在线题库匹配答案并逐格填写。

---

## 🌟 核心亮点

- **极致轻量**：主脚本仅 **10 KB**，去掉臃肿的内置题库。
- **在线加载**：一键从 GitHub 在线下载/更新最新题库到浏览器本地，支持断网离线使用。
- **一键清理**：提供「清理本地题库」按钮，随时清空浏览器本地缓存，干净无残留。
- **双模支持**：支持 SpreadJS 工作底稿自动填写 + 普通单据表单自动填写。

---

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `WZW-Audit-tool.user.js` | **Tampermonkey 主脚本**，支持一键下载、清理及自动填表逻辑 |
| `tiku.json` | **合并版在线题库**，包含全部 388 个索引数据（表单+单据答案） |
| `extract_all.py` | **全量提取脚本**：扫描 `tiku/` 文件夹下所有 xlsx + docx 文件并生成数据 |
| `regenerate_all.py` | **题库重组脚本**：将提取出的数据处理并进行合并重组 |

---

## ⚙️ 安装与使用方法

### 1. 安装脚本
1. 在浏览器中安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展。
2. 打开 `WZW-Audit-tool.user.js`，Tampermonkey 将自动识别并提示安装。

### 2. 下载题库
1. 安装脚本后，进入任意审计实训页面。
2. 页面右下角会出现三个浮动控制按钮：
   - **📥下载题库**：首次使用必点。点击后将自动从 GitHub 获取最新题库并保存在浏览器本地（`localStorage`）。
   - **🗑清理本地题库**：点击后可以随时清空浏览器中缓存的题库数据。
   - **🤖万能填表**：在题库下载完成后，点击即可自动填表。

---

## 🛠 重新提取与更新

若需要自己更新或扩充题库，可以使用目录下的 Python 提取工具：

```bash
python3 extract_all.py     # 全量扫描 tiku/ 文件夹下所有 xlsx 和 docx 文件
python3 regenerate_all.py   # 重组并生成最小化题库
```

---

## 💡 工作原理

1. **表格识别**：监听页面 SpreadJS 控件加载，在表格中自动搜索索引号（如 `DA1-3`, `ZA-1` 等）。
2. **偏移计算**：自动计算实际坐标与题库数据的偏移，完美适配不同模板的标题行。
3. **数据回填**：通过 API 逐格回填答案，并支持复选框/单选框布尔值及单据表单自动定位填写。
