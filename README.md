# 审计实训万能填表助手 (WZW-Audit-tool)

智能审计实践教学平台（http://10.18.0.178:9350）的用户脚本，自动识别 SpreadJS 表格，从题库匹配答案并逐格填写，同时支持单据表单自动填充。

---

## 核心亮点

- **双版本**：在线版（轻量按需下载） + 内置版（即装即用，无需网络）
- **国内友好**：内置版无需联网，在线版使用 jsDelivr CDN 加速，GitHub 受阻也可下载
- **脚本猫/油猴双兼容**：支持 [ScriptCat（脚本猫）](https://scriptcat.org/) 和 [Tampermonkey（油猴）](https://www.tampermonkey.net/)
- **双模支持**：SpreadJS 工作底稿自动填写 + 普通单据表单自动填写

---

## 安装方法

### 方式一：通过脚本猫（推荐，国内用户首选）

1. 安装 [ScriptCat（脚本猫）](https://scriptcat.org/) 浏览器扩展（国内可访问）
2. 访问 GitHub 仓库中的脚本文件：
   - **[在线版（推荐）](https://github.com/allen0039/WZW-Audit-tool/raw/main/WZW-Audit-tool.user.js)** — 轻量（18 KB），首次使用需点击「下载题库」
   - **[内置版（无需网络）](https://github.com/allen0039/WZW-Audit-tool/raw/main/WZW-Audit-tool-builtin.user.js)** — 完整（2.2 MB），388 条题库直接内置，即装即用
3. 脚本猫会自动识别并提示安装

### 方式二：通过油猴

1. 安装 [Tampermonkey（油猴）](https://www.tampermonkey.net/)（如无法访问可在国内应用商店搜索安装）
2. 同样打开上述脚本链接安装

### 方式三：手动安装

1. 复制对应脚本源的完整代码
2. 在脚本猫/油猴的管理面板中新建脚本，粘贴保存

## 使用方法

1. 安装脚本后，进入审计实训平台（http://10.18.0.178:9350）
2. 页面右下角会出现悬浮面板：
   - **在线版**：先点击「📥下载题库」（jsDelivr CDN 自动加速），成功后状态变为绿色
   - **内置版**：安装后直接可用，无需任何下载操作
3. 在需要填写的表格/单据页面，点击「🤖 自动填充表格/单据」
4. 脚本会自动识别索引号、计算偏移、填入答案

---

## 工作原理

1. **表格识别**：监听页面 SpreadJS 控件，自动搜索索引号（如 `DA1-3`、`ZA-1` 等）
2. **偏移计算**：自动计算实际坐标与题库数据的偏移，适配不同模板标题行
3. **数据回填**：逐格回填答案，支持复选框/单选框布尔值及单据表单自动填写
